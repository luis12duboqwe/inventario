# Centro de capacitación Softmobile v2.2.0

Esta carpeta consolida manuales imprimibles (PDF) y guías rápidas en video para cada módulo operativo. Los archivos se generan en español y se mantienen alineados con el modo estricto de versión vigente.

## Material disponible

- `manual_inventario.pdf` y `videos/inventario_resumen.txt`: flujo de altas, auditorías y reportes de inventario.
- `manual_operaciones.pdf` y `videos/operaciones_resumen.txt`: POS, compras, devoluciones, transferencias y cierres.
- `manual_analytics.pdf` y `videos/analytics_resumen.txt`: tableros, filtros y exportaciones en tema oscuro.
- `manual_seguridad.pdf` y `videos/seguridad_resumen.txt`: autenticación, 2FA opcional y bitácora de auditoría.

Cada manual PDF se deriva de un guion en texto plano incluido aquí mismo para facilitar su actualización sin herramientas externas. Los videos cortos se documentan con guiones narrativos (`*.txt`) listos para grabación o locución automática.
